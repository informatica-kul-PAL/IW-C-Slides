== Voorbeeld oefening
=== Opgave
De opgave is terug te vinden in `playlist.h`, je implementeert de functies in `playlist.c`
=== Tests
Er zijn enkele tests voorzien door ons in `pal_test.c`, het is de bedoeling dat je zelf ook testen bijschrijft in `student_test.c`
=== Runnen
Je kan de tests runnen met:
```
make run
```
